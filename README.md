# bytemines.io

Personal website and blog built with Next.js 16 and React 19.

## Requirements

- [Bun](https://bun.sh/) (package manager)
- Node.js 20+

## Quick Start

```bash
# Install dependencies
make install

# Start development server
make dev
```

Open [http://localhost:4321](http://localhost:4321) to view the site.

## Available Commands

| Command | Description |
|---------|-------------|
| `make dev` | Start development server (port 4321) |
| `make build` | Build for production |
| `make start` | Start production server |
| `make stop` | Stop production server |
| `make restart` | Restart production server |
| `make status` | Check if server is running |
| `make rebuild` | Clean and rebuild |
| `make clean` | Remove build artifacts |
| `make lint` | Run ESLint |
| `make type-check` | Run TypeScript type checking |
| `make check` | Run all quality checks |
| `make install` | Install dependencies |
| `make update` | Update dependencies |
| `make fresh` | Clean install and rebuild |
| `make help` | Show all available commands |

## Tech Stack

- [Next.js 16](https://nextjs.org/) - React framework
- [React 19](https://react.dev/) - UI library
- [Tailwind CSS 4](https://tailwindcss.com/) - Styling
- [MDX](https://mdxjs.com/) - Blog content
- [Bun](https://bun.sh/) - Package manager & runtime
