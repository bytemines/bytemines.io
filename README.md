# bytemines.io

Personal website and blog built with Next.js 16 and React 19.

## Quick Start

```bash
make dev
```

Opens at [http://localhost:4321](http://localhost:4321). Dependencies install automatically.

## Commands

```
make dev     Start development server
make build   Build for production
make start   Start production server
make stop    Stop server
make clean   Clean build files
make lint    Run linter
```

## Tech Stack

Next.js 16 / React 19 / Tailwind CSS 4 / MDX / Bun
